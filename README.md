# Dori
Face detecting doll -> Dori


 ### 이름 : 도리도리짝짝
 ### 팀명 : 도리도리짝짝 ( 멤버 : 전찬혁 )
 ### 개발환경 : OpenCV 2.4.13, C++, Visual Studio 2017, Arduino
 ### 프로젝트 : 얼굴을 따라가는 귀여운 '도리'가 사람의 얼굴을 인식해 따라오며 날개로 귀여운 제스쳐를 취합니다
 -> 아래부분에 도리 소개 있습니다 !
